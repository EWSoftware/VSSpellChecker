Changes made in this release:


- Fixed incorrect path resolution for additional dictionary folders and the global ignored words
file.
- Set the ignore private identifiers, internal identifiers, compiler generated code, and member
body identifiers code analyzer options to true to limit spell checking to public and protected identifiers by
default.
- Reordered the configuration editor categories to list the general and code analyzer categories
first.
